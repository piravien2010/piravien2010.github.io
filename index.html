LA 7710

Was ist jquery

jQuery ist eine JavaScript-Bibliothek, die Webentwicklern die Arbeit mit JavaScript erleichtern soll.
Sie überbrückt technische Unterschiede zwischen verschiedenen Browsern und ermöglicht komfortablen und kompakten Code,
der im direkten Vergleich zu JavaScript häufig sehr viel kürzer ist.
